Customer Support Ticket Visualization & Insights 

Executive Summary:
This project analyzes customer support interactions to categorize common pain points. By quantifying ticket types, businesses can allocate resources efficiently to improve customer satisfaction.

​Data Insights:
​Primary Concern: "Bug" reports constitute the largest portion of incoming tickets. 
​Feature Requests: Customers are actively suggesting new improvements, indicating high user engagement. 

Technical Implementation:
Exploratory Data Analysis (EDA): Used Pandas value_counts() to aggregate categorical data. 
​Visualization: Built a categorical bar plot using Matplotlib to present a clear "Insights + Charts" report.
